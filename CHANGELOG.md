# Changelog

All notable changes to ClipBox are documented in this file.

## [0.1.2] - 2026-06-29

### 新增
- 回收站：删除先进回收站，默认保留 24 小时，支持撤销恢复
- 设置中「合并历史重复项」，一键清理已有重复记录
- 面板可在光标附近弹出，多显示器自动落在当前屏幕
- 完整快捷键说明（↑↓、Enter、Shift+Enter、1–9、Esc）
- 图片缩略图懒加载，减轻长列表内存压力
- 历史接近上限时提示

### 改进
- Shift+Enter 纯文本粘贴（无需再按 Ctrl）
- 键盘选择条目时自动滚动到可见区域
- 删除后 8 秒内可一键撤销

## [0.1.1] - 2026-06-26

### 新增
- 关键内容胶囊：URL / 文件路径 / 验证码 / 快递单号 / JSON / 代码
- 链接类胶囊点击打开浏览器或文件；JSON / 代码胶囊一键格式化后粘贴
- 右键菜单与行内 `{ }` 按钮：格式化后粘贴
- 来源应用图标（Windows exe 图标缓存）
- 自然语言搜索：昨天 / 今天 / 本周、应用名（Chrome、豆包等）、`#url` `#code` `#img`
- 中文搜索改用子串匹配，修复搜不到中文关键词的问题
- 面板位置记忆、虚拟列表滚动优化

### 改进
- 胶囊智能折叠：有链接时默认只显示链接 + `▸` 展开按钮，避免误触
- 收紧代码识别规则，长段中文不再误判为代码
- 数据库级去重：相同内容再次复制会置顶已有条目，不再重复入库
- 深色模式下胶囊 / 应用头像对比度自适应
- 历史列表排版：更紧凑的工具栏、两行正文预览、操作按钮不遮挡文字
- 内存：高亮缓存 LRU、关闭面板可释放内存、图标缓存

### 修复
- 虚拟列表进入历史空白、滚动后才有内容
- 滚动时列表被弹回顶部
- 胶囊悬停显示整篇正文的大提示框
- 来源应用名改用进程名，图标匹配更准确

## [0.1.0] - 2026-06-26

### 新增
- 剪贴板历史：文本 / HTML / 图片 / 文件路径
- 固定片段、分类筛选、FTS 搜索
- Win+V 风格单击粘贴、模拟 Ctrl+V
- 中英文界面、设置面板（主题、热键、保留策略等）
- Windows / Linux CI 自动构建
