# Changelog

All notable changes to ClipBox are documented in this file.

## [Unreleased]

## [0.1.3] - 2026-07-02

### 新增
- **敏感信息自动打码**：识别身份证、银行卡、手机号、验证码、密钥等，列表与通知预览自动隐藏；粘贴/复制仍用原文，支持 👁 临时预览
- 设置 → 隐私与性能：敏感打码开关、OCR 开关（**默认关闭**）、批量补全已有图片 OCR
- 设置内**检查更新**：从 GitHub Release 拉取，支持一键下载安装并重启
- **存储详情面板**：分项占用、最大条目、深度清理、打开数据目录
- **大图自动压缩**：超过阈值/尺寸存 JPEG，可配置质量与最长边
- **Windows 截图 OCR**：开启后后台识别，截图文字可搜索；条目上可手动点「文」识别

### 改进
- 界面质感：列表过渡动画、操作按钮淡入、芯片 hover 反馈；前 9 条显示快捷键数字时隐藏 T/图 类型标签，减少拥挤
- 代码格式化按钮改为 SVG 图标；拖动条与窗口顶部一体化（去掉分隔线）
- OCR 逻辑：跳过过小图片与噪声结果；手动识别与批量补全
- 内存：缩略图/图标 LRU 缓存、缩小虚拟列表预渲染范围、关闭面板时释放缩略图缓存

## [0.1.2] - 2026-06-29

### 新增
- 回收站：删除先进回收站，默认保留 24 小时，支持撤销恢复
- 设置中「合并历史重复项」，一键清理已有重复记录
- 面板可在光标附近弹出，多显示器自动落在当前屏幕
- 完整快捷键说明（↑↓、Enter、Shift+Enter、1–9、Esc）
- 图片缩略图懒加载，减轻长列表内存压力
- 历史接近上限时提示

### 改进
- Shift+Enter 纯文本粘贴（无需再按 Ctrl）
- 键盘选择条目时自动滚动到可见区域
- 删除后 8 秒内可一键撤销

## [0.1.1] - 2026-06-26

### 新增
- 关键内容胶囊：URL / 文件路径 / 验证码 / 快递单号 / JSON / 代码
- 链接类胶囊点击打开浏览器或文件；JSON / 代码胶囊一键格式化后粘贴
- 右键菜单与行内 `{ }` 按钮：格式化后粘贴
- 来源应用图标（Windows exe 图标缓存）
- 自然语言搜索：昨天 / 今天 / 本周、应用名（Chrome、豆包等）、`#url` `#code` `#img`
- 中文搜索改用子串匹配，修复搜不到中文关键词的问题
- 面板位置记忆、虚拟列表滚动优化

### 改进
- 胶囊智能折叠：有链接时默认只显示链接 + `▸` 展开按钮，避免误触
- 收紧代码识别规则，长段中文不再误判为代码
- 数据库级去重：相同内容再次复制会置顶已有条目，不再重复入库
- 深色模式下胶囊 / 应用头像对比度自适应
- 历史列表排版：更紧凑的工具栏、两行正文预览、操作按钮不遮挡文字
- 内存：高亮缓存 LRU、关闭面板可释放内存、图标缓存

### 修复
- 虚拟列表进入历史空白、滚动后才有内容
- 滚动时列表被弹回顶部
- 胶囊悬停显示整篇正文的大提示框
- 来源应用名改用进程名，图标匹配更准确

## [0.1.0] - 2026-06-26

### 新增
- 剪贴板历史：文本 / HTML / 图片 / 文件路径
- 固定片段、分类筛选、FTS 搜索
- Win+V 风格单击粘贴、模拟 Ctrl+V
- 中英文界面、设置面板（主题、热键、保留策略等）
- Windows / Linux CI 自动构建
